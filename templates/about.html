About the data set

This data set contains record for deaths due to opioids of four classes , prescription, syntehtic, herion, cocaine. As well deaths by specific sex and race. This data range is from 1999-2019. The data used is this web app is from the CORGIS Dataset Project and can be found here.

An example of one of the statistics reported each year is deaths due to synthetic opioids. A synthetic opioid is any opioid synthesized in a lab. for example fentanyl is a synthetic opioid.( source)
