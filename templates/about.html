About the data set

This data set contains record for deaths due to opioids by state. This data range is from 1999-2016. The data used is this web app is from the NCHS - Drug Poisoning Mortality by State. All data is from the United States and can be found here.

An example of one of the statistics reported each year is Crude rate. A crude rate is the number of new cases (or deaths) occurring in a specified population per year. The popultation is 100,000. ( source)
